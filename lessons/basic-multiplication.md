# Author Notes

This lesson should help a child understand:

- multiplication means equal groups
- multiplication is repeated addition
- the first number tells how many groups
- the second number tells how many are in each group

Keep the tone concrete and visual. Use stars, blocks, apples, or toy groups when answering pause-point questions.
